# Travel Keyboard

A compact mechanical keyboard designed in **KiCad** with portability and travel in mind. This project focuses on creating a reliable, lightweight, and customizable keyboard that can easily fit into a backpack or travel setup while providing a comfortable typing experience wherever you go.

---

## Overview

Traveling often means working from different locations such as airports, cafés, hotels, libraries, or coworking spaces. Carrying a full-size keyboard can be inconvenient, while laptop keyboards are not always comfortable for extended use.

This project aims to provide a compact mechanical keyboard that balances portability, durability, and usability. The entire PCB was designed from scratch using KiCad, making it suitable for customization and future improvements.

---

## Features

* Compact layout for easy portability
* Custom PCB designed in KiCad
* Mechanical switch support
* Individual diode for every key
* USB connectivity
* Lightweight and travel-friendly design
* Easy to assemble and customize
* Suitable for remote work, study, and travel

---

## Design

The PCB was created using KiCad by designing the schematic first and then converting it into the PCB layout. Every key is paired with its own diode to enable proper matrix scanning and reduce the possibility of ghosting.

The design emphasizes:

* Efficient component placement
* Clean routing
* Compact board dimensions
* Reliable electrical connections
* Ease of manufacturing

## Why a Travel Keyboard?

A travel keyboard should be:

* Small enough to fit in a backpack
* Durable enough for frequent transport
* Comfortable for long typing sessions
* Easy to repair and customize

This project combines these goals into a compact PCB that is practical for everyday travel and mobile work environments.
